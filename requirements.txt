# DEPENDÊNCIAS EXTERNAS (Ferramentas do Sistema):
# ==================================================
# Certifique-se de instalar estas ferramentas no seu sistema:
#
# - RAR: Compactador de arquivos
#   Ubuntu/Debian: sudo apt-get install rar
#   Fedora/RHEL:   sudo dnf install rar
#   macOS:         brew install rar
#
# - PAR2 (ou parpar): Gerador de paridade
#   Ubuntu/Debian: sudo apt-get install par2
#   Fedora/RHEL:   sudo dnf install par2cmdline-mt
#   macOS:         brew install par2
#   Alt (mais rápido): npm install -g parpar
#
# - Nyuu: Cliente upload Usenet
#   npm install -g nyuu
#   Ou compilar: https://github.com/Piorosen/nyuu
